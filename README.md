**persyaratan:**
 * pyqrcode
 * setuptools
 * whell
 
 
**Install persyaratan:**
```bash
$ pip install -r syarat.txt
```
**INSTALL**
```bash
$ git clone https://github.com/bimasena321/QR_encode/
$ cd QR_encode
```
**penggunaan untuk Windows dan Mac os:**
```bash
$ python QR.py
```
**penggunaan untuk Linux:**
```bash
$ python3 QR.py
```
